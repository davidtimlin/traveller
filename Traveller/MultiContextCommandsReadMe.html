Reference commands for multiple EF contexts in the same project.

IdentityContext

  • Enable migrations:
    Enable-Migrations -ContextTypeName TravellerMVC.DataContexts.User.UserDbContext -MigrationsDirectory DataContexts\User\Migrations
  • Add migration:
    Add-Migration -ConfigurationTypeName TravellerMVC.DataContexts.User.Migrations.Configuration -Name "NAME"
  • Update database:
    Update-Database -ConfigurationTypeName TravellerMVC.DataContexts.User.Migrations.Configuration

TravellerContext

  • Enable migrations:
    Enable-Migrations -ContextTypeName TravellerMVC.DataContexts.Traveller.TravellerDbContext -MigrationsDirectory DataContexts\Traveller\Migrations
  • Add migration:
    Add-Migration -ConfigurationTypeName TravellerMVC.DataContexts.Traveller.Migrations.Configuration -Name "NAME"
  • Update database:
    Update-Database -ConfigurationTypeName TravellerMVC.DataContexts.Traveller.Migrations.Configuration
